# Jesse-Joel Nzumafor | Portfolio

A professional portfolio website for Jesse-Joel Nzumafor, Digital Marketing & Strategy Specialist.

## Features

- **Single-page design** with smooth scrolling navigation
- **Responsive layout** for desktop, tablet, and mobile
- **Dark theme** with amber accent colors
- **Sections**: Hero, About, Experience, Skills, Contact
- **Lightweight** – vanilla HTML, CSS, and JavaScript

## Getting Started

### Option 1: Quick preview (recommended)
```bash
npm run preview
```
Then open **http://localhost:5500** in your browser.

### Option 2: Live Server (VS Code / Cursor)
1. Install the "Live Server" extension if you haven't
2. Right-click `index.html` → **Open with Live Server**
3. The site opens in your browser automatically

### Option 3: Open file directly
Double-click `index.html` to open it in your default browser. (Some features like fonts may load differently.)

2. **Deploy**: Upload the folder to any static hosting (Netlify, Vercel, GitHub Pages, etc.) or a web server.

## Structure

```
Portfolio website/
├── index.html      # Main page
├── styles.css      # All styles
├── script.js       # Navigation & scroll effects
├── assets/         # Images (profile photo)
└── README.md
```

## Customization

- **Profile photo**: Replace `assets/profile.png` with your preferred image.
- **Contact links**: Update LinkedIn URL in `index.html` (search for `linkedin.com`).
- **Resume**: Add a PDF link to the Resume contact card or upload `Jesse-Joel Resume.pdf` to `assets/` and link it.
